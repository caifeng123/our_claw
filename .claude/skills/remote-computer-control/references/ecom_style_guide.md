# 电商主图构图策略指南

## 适用产品线
Sanrio（三丽鸥）IP 系列产品：
- 潮玩摆件：光动乐、捏捏乐、场景摆件、盲盒手办
- 卡通实用物件：手持风扇、卡通手表、卡通抓夹/发饰、日用品

## 通用原则

### 背景策略（不使用白底）
| 优先级 | 背景类型 | 说明 | 示例 |
|--------|---------|------|------|
| 首选 | 角色主题色/渐变 | 匹配角色代表色，营造 IP 氛围 | Hello Kitty=柔粉、My Melody=薰衣草紫、Kuromi=黑紫渐变、Cinnamoroll=天蓝、Pompompurin=暖黄 |
| 备选 | 生活场景 | 桌面/书架/梳妆台，营造真实使用感 | 木纹书桌、粉色梳妆台、少女房间一角 |
| 备选 | 纯色柔和背景 | 马卡龙色系、奶油色，柔和不抢焦 | 奶油黄、薄荷绿、浅珊瑚 |
| 避免 | 纯白底 | 缺乏氛围感，不适合潮玩品类 | — |

### 光影风格
- **首选**：柔和漫射光，营造治愈可爱感
- **可选**：侧逆光 + 柔焦，突出材质质感（尤其光动乐发光效果）
- **避免**：硬光直打（产生浓重阴影，破坏可爱氛围）

### 角色辨识度
- 角色面部/标志性特征必须清晰可见，不可被裁切或遮挡
- 多角色产品需展示所有角色，确保主角色在视觉焦点位置

## 品类构图策略

### 一、光动乐（发光/动态摆件）

核心卖点：**发光效果 + 角色造型 + 动态感**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 发光氛围图 | 微暗环境中产品发光，光晕扩散，营造梦幻感 | **默认首选** — 最能体现光动乐核心卖点 |
| 场景摆放图 | 放置于书桌/床头柜/书架，自然光 + 发光效果叠加 | 展示使用场景和尺寸感 |
| 多角度组合 | 2-3 个角度拼合（正面+侧面+发光对比） | 展示完整造型 |

**Prompt 要素**：
- 必须体现发光效果（glow, LED illumination, soft light emission）
- 背景微暗或使用角色主题色渐变，衬托光效
- 强调材质质感（透明/半透明部件的光线穿透感）

### 二、捏捏乐（软胶解压玩具）

核心卖点：**软弹手感 + 角色可爱造型 + 解压趣味**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 手持挤捏图 | 手指轻捏产品，展示形变和软弹质感 | **默认首选** — 直观传达手感 |
| 排列展示图 | 多款并排/散落排列，丰富色彩 | 系列产品展示 |
| 特写质感图 | 微距拍摄，展示表面纹理和材质细节 | 强调品质感 |

**Prompt 要素**：
- 手持图必须展示自然的手指按压和材质形变（squeezing, deformation, soft squishy texture）
- 强调材质：慢回弹（slow-rising）、Q弹（bouncy, squishy）
- 色彩饱满，角色表情清晰可见

### 三、卡通手持风扇

核心卖点：**角色造型 + 便携实用 + 夏日清凉感**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 手持使用图 | 手握风扇，微风吹拂感（发丝/丝带飘动暗示） | **默认首选** — 展示使用场景 |
| 产品正面特写 | 展示角色造型设计和风扇整体 | 强调造型设计 |
| 夏日场景图 | 户外/海边/公园场景，搭配夏日元素 | 季节营销 |

**Prompt 要素**：
- 传达清凉感（breeze, wind effect, summer freshness）
- 角色造型作为风扇主体需完整展示
- 背景使用清凉色调（天蓝、薄荷绿）或夏日场景

### 四、卡通手表

核心卖点：**角色造型表盘 + 佩戴可爱感 + 日常实用**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 手腕佩戴图 | 纤细手腕佩戴，展示表盘角色图案 | **默认首选** — 展示佩戴效果 |
| 表盘特写 | 微距展示表盘设计细节、指针、角色图案 | 强调设计细节 |
| 平铺搭配图 | 手表搭配同系列配饰，场景化展示 | 系列搭配 |

**Prompt 要素**：
- 表盘角色图案必须清晰可辨（clear character design on dial）
- 表带材质和颜色需呈现准确
- 佩戴图需暗示少女/儿童手腕（尺寸参照）

### 五、卡通抓夹 / 发饰

核心卖点：**角色造型 + 佩戴效果 + 精致小巧**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 发型佩戴图 | 模特后脑/侧面，抓夹固定在发髻上 | **默认首选** — 展示实际佩戴效果 |
| 手持展示图 | 手指捏住抓夹，展示造型和尺寸 | 强调造型细节 |
| 散落排列图 | 多款抓夹散落在梳妆台/镜子旁 | 系列展示 |

**Prompt 要素**：
- 佩戴图需展示抓夹在发型中的实际效果
- 角色造型细节清晰（character details clearly visible）
- 背景使用柔和色调，可搭配梳妆台/镜面元素

### 六、其他卡通日用品（水杯/钥匙扣/手机壳等）

核心卖点：**角色 IP + 日常使用场景 + 质感**

| 方案 | 构图方式 | 何时选择 |
|------|---------|---------|
| 使用场景图 | 自然使用状态（手握水杯/钥匙扣挂包上） | **默认首选** |
| 产品特写 | 45° 角展示完整造型和材质 | 强调产品本身 |
| 搭配氛围图 | 与同 IP 产品组合摆放 | 系列/套装营销 |

**Prompt 要素**：
- 根据具体品类调整场景（水杯→咖啡厅/桌面，钥匙扣→包包旁）
- 材质质感清晰（金属光泽/亚克力透明/硅胶软弹）
- 角色造型完整展示

## Sanrio 角色色彩速查

Agent 识别角色后，用此表匹配背景主题色：

| 角色 | 代表色 | 推荐背景渐变 |
|------|--------|-------------|
| Hello Kitty | 红+粉 | 柔粉→白粉渐变 |
| My Melody | 粉+薰衣草 | 浅粉→薰衣草紫渐变 |
| Kuromi | 黑+紫 | 深紫→黑紫渐变 |
| Cinnamoroll | 天蓝+白 | 天蓝→浅蓝白渐变 |
| Pompompurin | 黄+棕 | 暖黄→奶油色渐变 |
| Pochacco | 白+蓝 | 浅蓝→白渐变 |
| Tuxedo Sam | 蓝+白 | 冰蓝→白渐变 |
| Badtz-Maru | 黑+黄 | 深灰→黑渐变 |
| Little Twin Stars | 粉+蓝 | 粉蓝→淡紫梦幻渐变 |
| Hangyodon | 绿+蓝 | 海洋蓝绿渐变 |

> 若产品包含多角色，选主角色的代表色为主色调，
> 副角色代表色作为点缀或渐变过渡色。
